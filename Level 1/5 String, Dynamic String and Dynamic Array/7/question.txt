Remove Primes

1. You are given an ArrayList of positive integers.
2. You have to remove prime numbers from the given ArrayList and return the updated ArrayList.

Note -> The order of elements should remain same.
Input Format
A number N
arr1
arr2.. N numbers
Output Format
An Arraylist

Constraints
1 <= N <= 10000
Sample Input
4
3 12 13 15
Sample Output
[12, 15]
